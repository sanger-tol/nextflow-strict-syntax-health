Input/output options
  --input                   [string]  Path to comma-separated file containing information about the samples in the experiment. 
  --ancestral_table         [string]  TSV containing mappings for ancestral elements to busco genes. 
  --ancestral_busco_lineage [string]  The busco lineage used for ancestral painting (must match --ancestral_table). 
  --binsize                 [integer] Bin size in base pairs for cooler cload [default: 1000] 
  --kmer_size               [integer] Size for Fastk to create the k-mer library [default: 31] 
  --assembly                [string]  The Genbank assembly accession for the assembly, for example: GCA_922984935.2. 
  --biosample_wgs           [string]  The biosample accesion(s) linked to the WGS samples in the experiment, for example: SAMEA7520803. 
 
  --biosample_rna           [string]  The biosample accesion(s) linked to the RNA samples in the experiment, for example: SAMEA7521081. 
 
  --biosample_hic           [string]  The biosample accesion(s) linked to the Hi-C samples in the experiment, for example: SAMEA7520846. 
 
  --busco_lineage           [string]  Override the BUSCO database used for the assembly assessment. 
  --select_contact_map      [string]  Select which contact maps should be generated, higlass, pretext or both  (accepted: higlass, 
pretext, both) [default: both]  
  --btk_location            [string]  Location of the local blobtoolkit "blobdir" 
  --btk_online_location     [string]  Location of an online blobtoolkit "blobdir" 
  --outdir                  [string]  The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure. [default: results]  
  --email                   [string]  Email address for completion summary. 
  --multiqc_title           [string]  MultiQC report title. Printed as page header, used for filename if not otherwise specified. 
  --cool_order              [string]  Path to a file that contains the sequence names in the order you want to represent them on the 
contact-map.  
  --genome_notes_api        [string]  URL for Genome Notes Portal API . 
  --note_template           [string]  The path to a genome note template file. 
  --annotation_set          [string]  Path to annotation file :gff. 

Reference genome options
  --fasta                   [string] Path to FASTA genome file. 

Databases
  --lineage_db              [string] Local directory where clade-specific BUSCO lineage datasets are stored. 
  --lineage_tax_ids         [string] Local file that holds a mapping between BUSCO lineages and taxon IDs. 

Generic options

  --help_full               [boolean]         Display the full detailed help message. 
  --show_hidden             [boolean]         Display hidden parameters in the help message (only works when --help or --help_full are provided).
