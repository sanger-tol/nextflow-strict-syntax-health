Input/output options
  --input                             [string] Path to YAML file describing the input data. 
  --outdir                            [string] The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --email                             [string] Email address for completion summary. 

Assembly options
  --enable_metamdbg                   [boolean] Enable assembly with metaMDBG. [default: true] 

Assembly QC options
  --enable_rrna_prediction            [boolean] Predict ribosomal RNA (rRNA) genes in assemblies using Infernal and the included Rfam covariance 
models. [default: true]  
  --rfam_rrna_cm                      [string]  Covariance model for rRNA preditiction from RFAM. [default: ${projectDir}/assets/rRNA.cm] 
  --enable_genomad                    [boolean] Enable Genomad for classification of circular contigs for determination of plasmid status. 
[default: true]  
  --genomad_db                        [string]  Path to a directory containing a Genomad database. If not provided, the database will be downloaded 
automatically.  
  --genomad_splits                    [integer] The number of chunks to split the Genomad database into when running searches. Increasing this value 
decreases memory usage at the expense of runtime. [default: 1]  

Read mapping options
  --long_read_mapping_reads_per_chunk [integer] When mapping long reads, how many long reads to map to the assembly in a single chunk. [default: 
1000000]  
  --hic_aligner                       [string]  Which Hi-C aligner to use - either bwa-mem2 or minimap2.  (accepted: bwamem2, minimap2) 
[default: bwamem2]  
  --hic_mapping_cram_bin_size         [integer] When mapping Hi-C, map each CRAM file in slices of this size. [default: 10000] 
  --hic_mapping_minq                  [integer] Minimum map quality for a Hi-C read to be passed on for binning by Bin3C or Metator. [default: 
10]  

Binning options
  --enable_binning                    [boolean] Enable binning of metagenomes into genome bins. [default: true] 
  --enable_metabat2                   [boolean] Enable binning with Metabat2. [default: true] 
  --enable_maxbin2                    [boolean] Enable binning with MaxBin2. [default: true] 
  --enable_bin3c                      [boolean] Enable binning with Bin3C. [default: true] 
  --enable_metator                    [boolean] Enable binning with MetaTor. [default: true] 
  --minimum_contig_size               [integer] The minimum size of contig to be considered when binning. This option applies to MetaBat2, MaxBin2, 
and Bin3C, but not Metator. [default: 3000]  
  --minimum_hifi_perc_identity        [integer] The minimum required percent identity of a read mapping to the assembly to be counted during 
coverage estimation for MetaBat2 and MaxBin2. [default: 99]  
  --minimum_bin_size                  [integer] The minimum size of a bin, in basepairs, to be emitted. This parameter is currently respected by 
MetaBat2 and Bin3C, but not Metator or MaxBin2. [default: 150000]  

Bin refinement options
  --enable_bin_refinement             [boolean] Enable refinement of genome bins. [default: true] 
  --enable_dastool                    [boolean] Enable bin refinement using DAS_Tool. [default: true] 
  --enable_magscot                    [boolean] Enable bin refinement using MagScoT. [default: true] 
  --hmm_gtdb_pfam                     [string]  Path to the GTDB r207 Pfam HMM database. [default: 
${projectDir}/assets/gtdbtk_rel207_Pfam-A.hmm]  
  --hmm_gtdb_tigrfam                  [string]  Path to the GTDB r207 TigrFam HMM database. [default: 
${projectDir}/assets/gtdbtk_rel207_tigrfam.hmm]  

Bin QC options
  --enable_binqc                      [boolean] Enable quality checks of genome bins. [default: true] 
  --enable_checkm2                    [boolean] Enable QC using CheckM2. [default: true] 
  --checkm2_db                        [string]  Path to a local diamond database file for CheckM2. If not provided, the CheckM2 database will 
automatically be downloaded.  
  --enable_trnascan_se                [boolean] Predict tRNAs in genome bins using tRNAscan-SE. [default: true] 

Bin taxonomy options
  --enable_taxonomy                   [boolean] Enable taxonomic classification of genome bins. [default: true] 
  --enable_gtdbtk                     [boolean] Enable taxonomic classification with GTDB-Tk. [default: true] 
  --gtdbtk_db                         [string]  Path to a directory containing the GTDB-Tk database. 
  --gtdbtk_skip_ani_screen            [boolean] Skip the ANI screen check with skani when classifying genomes with GTDB-Tk. 
  --gtdbtk_use_full_tree              [boolean] Use the full bacterial phylogenetic tree when classifying bacterial genomes. This is enabled by 
default in the test profiles as it is required to use the mock database for testing. Enabling this 
with the full GTDB database requires a minimum of 320GB of memory for the GTDB-TK classify process, 
but will place all bins into a single tree.  
  --gtdb_bac120_metadata              [string]  Path to the GTDB-Tk bac120 metadata file. [default: 
https://data.ace.uq.edu.au/public/gtdb/data/releases/release226/226.0/bac120_metadata_r226.tsv.gz] 
 
  --gtdb_ar53_metadata                [string]  Path to the GTDB-Tk ar53 metadata file. [default: 
https://data.ace.uq.edu.au/public/gtdb/data/releases/release226/226.0/ar53_metadata_r226.tsv.gz] 
 
  --gtdbtk_min_completeness           [integer] Minimum CheckM2 completeness required for a bin to be classified by GTDB-Tk. [default: 50] 
  --gtdbtk_max_contamination          [integer] Maximum contamination allowed CheckM2 completeness for a bin to be classified by GTDB-Tk. 
[default: 10]  

Summary options
  --completeness_score                [integer] Multiplier for completeness when scoring bins for the bin summary. Used in formula 
(compleneness_score * completeness - contamination_score * contamination). [default: 1]  
  --contamination_score               [number]  Multiplier for contamination when scoring bins for the bin summary. Used in formula 
(compleneness_score * completeness - contamination_score * contamination). [default: 0.5]  

Generic options
  --help                              [boolean, string] Display the help message. 
  --help_full                         [boolean]         Display the full detailed help message. 
  --show_hidden                       [boolean]         Display hidden parameters in the help message (only works when --help or --help_full are provided).
